# üìä Dynamic Salary Analysis Dashboard (Excel)

## Project Overview

This project presents a professional **Salary Analysis Dashboard** created using **Microsoft Excel**. The primary goal was to visualize key Human Resources (HR) compensation data and provide interactive filters to explore salary trends quickly.

This dashboard leverages the power of **Pivot Tables** to aggregate employee data and display insights related to salary distribution across various organizational dimensions.

## üéØ Dashboard Highlights

The interactive dashboard provides immediate answers to critical HR questions, including:

* **Total Compensation Metrics:** Total Salaries ($4.96M), Total Employees (500), Locations (7), and Departments (7).
* **Geographical Insights:** Ranking of Average Salary by Location (e.g., Indianapolis vs. Philadelphia).
* **Career Progression:** Visualization of the **Salary-Experience Correlation** (how salary changes with years of experience).
* **Demographic Breakdown:** Comparison of Average Salary by Education Level (BSc, Master, PhD) and by Department (Finance, IT, R&D, etc.).
* **Job Market Benchmarking:** Identification of the Top 10 highest-paying job roles.
